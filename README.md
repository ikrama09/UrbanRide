# 📖 About

UrbanRide is a modern Full Stack Car Rental Platform developed to simplify the vehicle rental experience.

The platform allows users to browse available vehicles, authenticate securely, reserve cars, manage bookings, and access their rental history through a responsive and intuitive interface.

The application follows modern software engineering principles with scalability, security, and performance in mind.
